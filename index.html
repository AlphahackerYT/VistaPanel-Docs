Installation

There are two methods to install VPT theme’s. You can use any method you like. If you get error’s by using first method then please use second method. However, first method is highly recommended.

Method#1

This method will guide you to use this theme without uploading files to save your hosting resources and provide better performance.

Open MOFH Panel

  1. First Open MyOwnFreeHost Panel.
  2. Enter You Email and Password.
  3. Click on Login button.

Navigate to cPanel Adverts

  1. After Successful Login.
  2. Found “Your Adverts” in the sidebar.
  3. Click on it.
  4. Drop-down menu will appear.
  5. Click on “Control Panel Adverts Control”.
  6. You will be navigated to Control Panel Adverts Control.

Select Domain

  1. A Page Will Open.
  2. Select Domain from there of which you want to change theme of vPanel.
  3. Click on “Add/Change Panel Adverts” button.

Adding Code

  1. After Selecting Domain
  2. A New Page will appear where you can add your advertising code.
  3. Add this code in the Top & Bottom Advert Area. 
    <link rel="stylesheet" type="text/css" href="https://cdn.vpthemes.win/theme-name/icon_spritemap.css" />
<link rel="stylesheet" type="text/css" href="https://cdn.vpthemes.win/theme-name/styles.css" />
  4. Replace theme-name with one of the theme available on our github repository i.e. gradient-blue and the url will become https://cdn.vpthemes.win/gradient-blue/styles.css
  5. After replacing and adding this code in the top advert area click on “Add/Change Advert Code”.
  6. Then re-perform selecting domain step.
  7. Then add the same code in the Bottom Advert Area and click on “Add/Change Advert Code”.
  8. Done!
  9. After that Logout from vPanel of your website and then re-login to the vPanel of your website.

You will see that your vPanel theme is successfully changed to Any of your selected theme (in our case theme is “gradient-blue”) without uploading files. Check Theme names here or on GitHub project.

Method#2

This method will guide you about full manual installation of any theme by uploading files on your own server.

We are using example of gradient-blue theme here. You can download all other themes from GitHub or VPTOfficial Website. At the place of theme-name there can be any theme name.

Extract .rar File

  1. Extract “theme-name.rar” file using winrar.
  2. Open “theme-name” folder.
  3. Folder Listing will be like the screenshot

Open Folder containing main files

  1. After Opening “theme-name” folder.
  2. Be sure to read “readme.txt” file.
  3. Navigate to “theme files” folder.
  4. Folder Listing will be like the screenshot

Upload files

  1. Upload all the files present in “theme files” folder to your website using “FTP” or “File Manager of Control Panel”. If you don’t upload all the files then it will cause problems.
  2. After uploading get url of styles.css and icon_spritemap.css. URL will be like 
    https://yourwebsite.com/styles.css
https://yourwebsite.com/icon_spritemap.css

    OR

    https://yourwebsite.com/yourdirectory/styles.css 
https://yourwebsite.com/yourdirectory/icon_spritemap.css

Open MOFH Panel

  1. Open MyOwnFreeHost Panel.
  2. Enter You Email and Password.
  3. Click on Login button.

Navigate to cPanel Adverts

  1. After Successful Login.
  2. Found “Your Adverts” in the sidebar.
  3. Click on it.
  4. Drop-down menu will appear.
  5. Click on “Control Panel Adverts Control”.
  6. You will be navigated to Control Panel Adverts Control.

Select Domain

  1. A Page Will Open.
  2. Select Domain from there of which you want to change theme of vPanel.
  3. Click on “Add/Change Panel Adverts” button.

Adding Code

  1. After Selecting Domain
  2. A New Page will appear where you can add your advertising code.
  3. Add this code in the Top & Bottom Advert Area. 
    <link rel="stylesheet" type="text/css" href="https://yourdomain/icon_spritemap.css" />
<link rel="stylesheet" type="text/css" href="https://yourdomain/styles.css" />
  4. Replace yourdomain with the url where you uploaded files
  5. After Adding this code in the top advert area click on “Add/Change Advert Code”.
  6. Then re-perform selecting domain step.
  7. Then add the same code in the Bottom Advert Area and click on “Add/Change Advert Code”.
  8. Done!
  9. After that Logout from vPanel of your website and then re-login to the vPanel of your website.
  10. Successfully Installed!

You will see that your vPanel theme is successfully changed to Any of your selected theme (in our case theme is “gradient-blue”) without uploading files. Check Theme names here or on GitHub project.

Code

You can use the below provided code for installation of themes & script’s. There are three different code’s for themes with two methods.

Normal Themes

Below you can find code for Normal Themes which mostly include only two files.

Method#1 Code

<link rel="stylesheet" type="text/css" href="https://cdn.vpthemes.win/theme-name/icon_spritemap.css" />
<link rel="stylesheet" type="text/css" href="https://cdn.vpthemes.win/theme-name/styles.css" />

Replace theme-name with any theme name provided here or on GitHub

Method#2 Code

<link rel="stylesheet" type="text/css" href="https://yourdomain/icon_spritemap.css" />
<link rel="stylesheet" type="text/css" href="https://yourdomain/styles.css" />

Replace yourdomain with your website where you had uploaded theme files.

Special Themes

Below you can find code for Special Themes which mostly include one or more than two files.

Method#1 Code

<link rel="stylesheet" type="text/css" href="https://cdn.vpthemes.win/theme-name/styles.css" />

OR

<link rel="stylesheet" type="text/css" href="https://cdn.vpthemes.win/theme-name/panel.css" />

Replace theme-name with any theme name provided here or on GitHub

Method#2 Code

<link rel="stylesheet" type="text/css" href="https://yourdomain/styles.css" />

OR

<link rel="stylesheet" type="text/css" href="https://yourdomain/panel.css" />

Replace yourdomain with your website where you had uploaded theme files.
